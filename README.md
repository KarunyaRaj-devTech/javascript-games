# JavaScript Games Portfolio 🎮

A collection of browser-based games built using **HTML, CSS, and JavaScript**.

This project demonstrates core concepts of game logic, DOM manipulation, and interactive UI design.

---

## Games Included

### 1️⃣ Tic Tac Toe
A classic two-player game where players take turns marking X and O on a 3×3 grid.

**Features**
- Player turn indicator
- Win detection
- Draw detection
- Winning cell highlight
- Restart button
- Modern UI with animations

**Technologies Used**
- HTML
- CSS
- JavaScript
