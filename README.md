# JavaScript Games Portfolio 🎮

A collection of browser-based games built using **HTML, CSS, and JavaScript**.

This project demonstrates core concepts of **game logic, DOM manipulation, event handling, animations, and interactive UI design**.

---

## Games Included

### 1️⃣ Tic Tac Toe

A classic two-player strategy game where players take turns marking **X** and **O** on a 3×3 grid.

**Features**
- Player turn indicator
- Win detection
- Draw detection
- Winning cell highlight
- Restart button
- Modern UI with animations

**Technologies Used**
- HTML
- CSS
- JavaScript

**Live Demo**

https://karunyaraj-devtech.github.io/javascript-games/tic-tac-toe/

---

### 2️⃣ Memory Card Game 🎴

A fun memory matching game where players flip cards to reveal emojis and try to find matching pairs.

**Features**
- 4×4 card grid
- Card flip animation
- Random card shuffle
- Match detection
- Move counter
- Win detection
- Restart game functionality

**Technologies Used**
- HTML
- CSS
- JavaScript

**Live Demo**

https://karunyaraj-devtech.github.io/javascript-games/memory-card-game/
